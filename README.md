# Totally Tremendous Cervice
TTC Service Alerter for iOS, macOS. (TTC subway only)

Requires iCloud Drive.

## Download
1. Go to [Releases](https://github.com/Dr-Sauce/TotallyTremendousCervice/releases/latest).
2. Click iCloud URL.
3. Press **Add Shortcut**.

## Setup
Launch **Shortcuts** app → Run **Totally Tremendous Cervice** → Press **Install** → **⨁ Add Shortcut** 

<img width="120" alt="IMG_0013" src="https://github.com/user-attachments/assets/88d2de38-1ce2-43f9-8dcf-5828fe8122d8" />
<img width="120" alt="IMG_0014" src="https://github.com/user-attachments/assets/6857efaf-99cc-4437-b5a3-4f8991dccdb7" />

**Automation** tab → **New Automation** → Select **App** → **App - `Choose`** → Select an app (or multiple) that you frequently use and press **✔️** → **Run Immediately** → **Next**
